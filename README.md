# Zawjati 🏡❤️

An intelligent AI companion that roleplays as a loving wife — offering warm conversations, emotional support, companionship, and daily life engagement.

## Features

- 🗣️ Natural conversation with personality
- 🧠 Long-term memory (preferences, habits, goals)
- 💬 Short-term conversation memory
- 🎭 Multiple personalities (gentle, playful, calm, energetic)
- 🌍 Multi-language support (English, Arabic, French)
- 📅 Daily check-ins and reminders
- 🎵 Music recommendations
- 📸 Image understanding
- ❤️ Emotion detection from messages
- ☁️ Secure local and cloud memory sync

## Project Structure

```
zawjati/

```
